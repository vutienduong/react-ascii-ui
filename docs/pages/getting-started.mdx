# Getting Started

Get up and running with React ASCII UI in minutes.

## Installation

Install the package via npm:

```bash
npm install react-ascii-ui
```

Or with yarn:

```bash
yarn add react-ascii-ui
```

## Basic Usage

Import and use components in your React application:

```tsx
import { AsciiButton, AsciiCard, AsciiInput } from 'react-ascii-ui';

function MyComponent() {
  return (
    <AsciiCard title="Login">
      <AsciiInput placeholder="Username" />
      <AsciiInput type="password" placeholder="Password" />
      <AsciiButton>Login</AsciiButton>
    </AsciiCard>
  );
}
```

## Styling

React ASCII UI uses Tailwind CSS classes for styling. Make sure you have Tailwind CSS configured in your project:

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

Update your `tailwind.config.js`:

```js
module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
    "./node_modules/react-ascii-ui/**/*.{js,jsx,ts,tsx}"
  ],
  theme: {
    extend: {
      fontFamily: {
        mono: ['ui-monospace', 'SFMono-Regular', 'Menlo', 'Consolas', 'monospace']
      }
    }
  },
  plugins: []
}
```

Add Tailwind to your CSS:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

## TypeScript Support

React ASCII UI is built with TypeScript and provides full type definitions. All components extend the appropriate HTML element props:

```tsx
import { AsciiButton } from 'react-ascii-ui';

// AsciiButton extends React.ButtonHTMLAttributes<HTMLButtonElement>
<AsciiButton 
  onClick={handleClick}
  disabled={isLoading}
  className="my-custom-class"
>
  Submit
</AsciiButton>
```

## Custom Styling

You can customize components by passing additional Tailwind classes:

```tsx
<AsciiButton className="text-red-400 border-red-400">
  Danger Button
</AsciiButton>

<AsciiCard className="bg-blue-900 border-blue-400" title="Custom Card">
  Content goes here
</AsciiCard>
```
