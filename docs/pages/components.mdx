## Component Props

All components extend their respective HTML element props and accept additional styling through the `className` prop.

### Common Props

- `className?: string` - Additional CSS classes
- `children?: React.ReactNode` - Child elements
- All standard HTML attributes for the respective element type

### Example with TypeScript

```tsx
import { AsciiButton } from 'react-ascii-ui';

interface Props {
  onSubmit: () => void;
  disabled?: boolean;
}

const SubmitButton: React.FC<Props> = ({ onSubmit, disabled }) => {
  return (
    <AsciiButton
      onClick={onSubmit}
      disabled={disabled}
      className="text-green-400"
      type="submit"
    >
      Submit Form
    </AsciiButton>
  );
};
```

## Customization

All components can be customized with Tailwind CSS classes:

```tsx
// Custom colors
<AsciiButton className="text-red-400 border-red-400 hover:bg-red-900">
  Danger
</AsciiButton>

// Custom spacing
<AsciiCard className="m-4 p-6" title="Spaced Card">
  Content with custom spacing
</AsciiCard>

// Custom fonts (if you want to override mono)
<AsciiAlert className="font-sans">
  Alert with custom font
</AsciiAlert>
```
